# Product Overview

First6 is a boutique, self-hosted NFL prediction tracking platform for private user groups.

## Core Features

- Track "First Touchdown" and "Anytime Touchdown" predictions
- Private user group management
- NFL game data ingestion and tracking
- Scoring and leaderboard system
- Multi-sport support architecture (NFL, NBA, MLB, NHL, CFB)

## Target Users

Private groups of friends tracking NFL touchdown predictions in a self-hosted environment.

## Deployment Model

Self-hosted via Docker Compose - designed for personal/small group deployment rather than cloud-scale SaaS.
